\section{Ergänzungen}

Neben den Wunschkriterien unter Abschnitt~\ref{sec:zielbestimmungen} sind derzeit
keine weiteren Ergänzungen geplant. Bisher ist von den \textit{Zielbestimmungen}
bis zum 16.5.2016 nur folgendes umgesetzt:

\begin{itemize}
  \item Neue Feeds werden oben in der Liste gezeigt.
  \item Ein Feed zeigt Titel, Datum, Teaser-Text und Link an.
  \item Die Feeds werden als Liste dargestellt.
  \item Die ActionBar hat ein Launcher-Icon und den App-Namen.
  \item Links werden im Browser geöffnet.
\end{itemize}
